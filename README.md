# Simskole Projekt

Detta dokument beskriver simskoleprojektet som syftar till att erbjuda simlektioner och aktiviteter för barn och vuxna. Målet är att förbättra simkunskaper och säkerhet i vatten genom professionell undervisning.

## Mål

- Att skapa en trygg och lärorik miljö för simlektioner.
- Att utveckla simkunskaper för alla åldrar.
- Att främja säkerhet i vatten genom utbildning.

## Lektioner

Vi erbjuder lektioner för olika åldersgrupper och färdighetsnivåer:
- Barnsim för nybörjare
- Fortsättningskurser för barn
- Vuxensimkurser
